
## Description

Basic CRUD operation using NestJs & PostgreSQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Routes implemented

```bash
# register a user
(POST)
http://localhost:3000/api/register

# login a user
(POST)
http://localhost:3000/api/login

# update a user
(PUT)
http://localhost:3000/api/users/:id

# delete a users
(DELETE)
http://localhost:3000/api/users/:id

# get all users
(GET)
http://localhost:3000/api/users
```
